📌 Overview

The Parking Lot Management System is a console-based Java application designed to simulate real-world parking operations. It allows users to park vehicles, remove them, and view the status of parking slots.

The project is built using Object-Oriented Programming (OOP) principles, making it easy to understand, extend, and maintain. It is ideal for beginners to practice Java concepts like classes, objects, and collections.

🚀 Features
Park a vehicle in the first available slot
Remove a vehicle using its vehicle number
Display all parking slots with current status
Simple menu-driven user interface
Handles edge cases:
Parking lot full
Vehicle not found
Efficient slot management using ArrayList
